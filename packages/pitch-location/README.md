# @ll-score/pitch-location

Reusable handedness-aware 3x3 strike-zone and special outside-zone input and
display. Pitch location is optional and remains independent of the umpire's
pitch result.

Planned internal structure:

```text
src/model/
src/application/
src/components/
src/index.ts
tests/
```

Shared geometry comes from `@ll-score/field-diagram`.
