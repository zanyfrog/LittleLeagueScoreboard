# @ll-score/hit-location

Reusable field-map input and display for batted-ball locations, normalized
coordinates, markers, and replay placement.

Planned internal structure:

```text
src/model/
src/application/
src/components/
src/index.ts
tests/
```

Selecting a location does not determine the scoring result. Shared geometry
comes from `@ll-score/field-diagram`.
