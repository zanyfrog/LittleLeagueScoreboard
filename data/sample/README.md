# Sample Data

Synthetic game and roster examples for development and documentation. Do not
place real child or family data here.

The canonical generated dataset is defined in `@ll-score/testing`. It includes:

- Springfield Little League and its 2026 Spring Season.
- Six teams.
- Eleven fictional players per team.
- Active team-season memberships and jersey numbers.
- Three games covering all six teams.
- Twenty-two-player roster snapshots for each game.

Load it with:

```powershell
pnpm seed:sample
```
